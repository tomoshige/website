    2. 扱う内容の理論の説明（定義・定理・数式を含む）

    今日は次回の連立方程式の話題で使用する概念である「階段行列」について説明します。今日扱う行列は、一般的な\(m \times n\)行列です。

    階段行列 (Row Echelon Form)

    • 定義:
      行列 \(A\) が階段行列であるとは、以下の条件を満たすときにいいます。
      1. ゼロ行（すべての成分が0の行）が存在する場合、これらはすべて、少なくとも1つはゼロではない行の下にある。
      2. 非ゼロ行の各行において、最初に現れる非ゼロ成分（ピボット）は、その行の左側にある全ての成分が0であり、かつ、その下の行のピボットは必ずその右側に位置する。

    簡約階段行列 (Reduced Row Echelon Form, RREF)

    • 定義:
      階段行列 \(A\) がさらに以下の条件を満たすとき、簡約階段行列と呼びます：
      1. 各ピボットは 1 である。
      2. 各ピボットの上および下のすべての成分は 0 である（すなわち、各ピボットはその列の唯一の非零成分である）。

    行基本変形による簡約階段行列の導出方法

    • 基本変形:

      • 行の入れ替え: 2つの行の順序を入れ替える。
      • 行の定数倍: 行全体をゼロでない定数で乗じる。
      • 行の加減: ある行に他の行の定数倍を加える（または引く）。

    • 操作フロー:

      1. 拡大係数行列（または任意の行列）を用意する。
      2. 前進消去を行い、左上から順に各行のピボット（最初の非ゼロ成分）を1に正規化し、その列の下部の成分を0にする。
      3. 後退消去を行い、各ピボットの上部の成分も0にして、簡約階段行列に変形する。

    行列のランクの定義

    • 定義:
      行列のランクとは、その行列の簡約階段行列において、ゼロでない行の個数を指します。
      • ランクは、その行列が持つ線形独立な行（または列）の最大数と同値です。
      • ランクは、連立一次方程式の解の一意性や、行列の可逆性の判定に重要な役割を果たします。

    3. 扱う内容の例（実例を提示、GPT, Colabなどは利用しない）

    例1: 階段行列と簡約階段行列の例

    • 階段行列の例:
      $$ A = \begin{pmatrix} 1 & 2 & -1 \\ 0 & 3 & 4 \\ 0 & 0 & 5 \end{pmatrix} $$ この行列は各行のリーディングエントリが右にずれており、ゼロ行が存在しないため階段行列です。

    • 簡約階段行列の例:
      $$ A_{\text{RREF}} = \begin{pmatrix} 1 & 0 & 2 \\ 0 & 1 & -3 \\ 0 & 0 & 0 \end{pmatrix} $$ ここでは、各リーディングエントリは1であり、各リーディングエントリ以外の同じ列の要素はすべて0です。

    例2: 行基本変形による簡約階段行列の導出

    • ある行列を基本変形により簡約階段行列に変形する例として、
      $$ B = \begin{pmatrix} 2 & 4 & -2 \\ 4 & 9 & -3 \\ -2 & -3 & 7 \end{pmatrix} $$ を考え、行の入れ替え、定数倍、行の加減を適用して RREF にする操作を行うことで、最終的にランクが2であることが確認できる（具体的な数値例は講義で口頭説明する）。

    例3: 行列のランクの定義の確認

    • ある行列が簡約階段行列に変形されたとき、非ゼロ行の個数がその行列のランクであることを確認します。
      例えば、上記の $ A_{\text{RREF}} $ の場合、非ゼロ行は2行なので、ランクは2です。

    4. ChatGPTによる解説＋Google Colab での実行例

    ChatGPTによる解説

    • 解説ポイント:
    • 連立一次方程式の解法において、行列のランクは解の存在や一意性の判断に非常に重要です。
    • 階段行列と簡約階段行列の変形を通じて、どの行が線形独立かを明確にでき、ランクはその数値として現れます。
    • この変形は、基本変形のルールに従って行われ、どの操作も行列のランクを変えないため、解の正しさが保証されます。

    Google Colabでの実装例

    以下は、ある行列に対して、基本変形を用いて簡約階段行列（RREF）に変形し、ランクを計算する Python のコード例です。途中経過も print で表示して、どのような手順でランクが求められるかを確認します。

    import numpy as np

def rref_verbose(A):
    """
    行列 A を簡約階段行列 (RREF) に変形し、その途中経過を print で表示する関数。
    入力:
      A: 任意の m x n 行列
    出力:
      RREF に変形された行列
    """
    A = A.copy().astype(float)
    m, n = A.shape
    row = 0
    for col in range(n):
        if row >= m:
            break
        # ピボット行の選択：col 列で非ゼロの要素がある行を探す
        pivot_row = None
        for r in range(row, m):
            if not np.isclose(A[r, col], 0):
                pivot_row = r
                break
        if pivot_row is None:
            continue
        # 行の入れ替え：現在の行 row と pivot_row を入れ替える
        A[[row, pivot_row]] = A[[pivot_row, row]]
        print(f"\n--- 行 {row} と 行 {pivot_row} の入れ替え ---")
        print(A)

        # 行の定数倍：ピボットを 1 にする
        pivot = A[row, col]
        A[row] = A[row] / pivot
        print(f"\n--- 行 {row} を {pivot:.3f} で割って正規化 ---")
        print(A)

        # 行の加減：ピボット行を使って、他の行の col 列を 0 にする
        for r in range(m):
            if r != row:
                factor = A[r, col]
                A[r] = A[r] - factor * A[row]
                print(f"\n--- 行 {r} から {factor:.3f}×行 {row} を引く ---")
                print(A)
        row += 1
    return A

# 例題：連立一次方程式の係数行列（拡大係数行列ではなく、純粋な係数行列でランクを計算）
# ここでは、講義例として以下の行列を考える：
# M = [ [3, 1, 1],
#       [4, 3, -1],
#       [5, 4, 1] ]
M = np.array([
    [3, 1, 1],
    [4, 3, -1],
    [5, 4, 1]
], dtype=float)

print("初期行列 M:")
print(M)

RREF = rref_verbose(M)
print("\n最終的な簡約階段行列 (RREF):")
print(RREF)

# ランクの計算：RREF の非ゼロ行の個数を数える
rank = np.sum(~np.all(np.isclose(RREF, 0), axis=1))
print(f"\n行列のランクは {rank} です。")

    5. 学んだ内容がどのように応用されるか

    • 連立方程式の解の判定:
      ランクを用いることで、連立一次方程式の解が一意であるか、無限解が存在するか、または解が存在しないかを判断できます。
    • 行列の可逆性:
      正方行列のランクがそのサイズと等しければ、その行列は可逆であると判定できます。
    • 統計学と機械学習:
      主成分分析（PCA）など、データの次元削減や特徴抽出において、データ行列のランクは重要な役割を果たします。
    • システムのモデリング:
      物理や工学のシステムにおいて、システムの状態や制約条件を表す行列のランクは、システムの自由度や冗長性の評価に利用されます。
